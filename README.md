# Landing Page Project

This project is part of The Odin Project Foundations course. The goal was to build a simple landing page using HTML and CSS while practicing layout techniques and basic styling.

## Overview
The landing page is a static website that includes:
- A header with navigation links  
- A hero section with main content and call-to-action  
- Information sections  
- A quote/testimonial section  
- A footer  

This project focuses on structuring content and using CSS to create a clean and responsive layout.

## Technologies Used
- HTML5  
- CSS3 (Flexbox)

## What I Learned
- Structuring a webpage using semantic HTML  
- Using Flexbox for layout and alignment  
- Styling elements with CSS  
- Organizing sections for better readability  
- Improving overall page design and spacing
